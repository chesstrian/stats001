Configuración para la Base de Datos

Base de Datos

Para usar esta Herramienta

  • Ingrese a Configuración y rellene los parámetros para conexión de la Base de Datos.

  • En el campo Extensión ingrese una extensión existente en el sistema.

  • En los campos siguientes (Desde - hasta), ingrese una fecha inicial y una final para los cuales desea conocer las llamadas de la extensión ingresada.

  • Dé click en el botón Mostrar para ver las estadísticas.

  • Seleccione lo que desea hacer con los datos en la tabla.
